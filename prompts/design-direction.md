# Design Direction Prompt

```text
Create 3 commercial-grade design directions for this product.

Read:
- docs/00-project-context.md
- docs/01-product-brief.md
- docs/02-prd.md

For each direction, provide:
- name
- best-fit product type
- audience fit
- visual style
- layout structure
- typography approach
- color approach
- component style
- risks
- why the user should choose it

The three directions should be:
1. Conservative/professional
2. Premium/product-led
3. Dense/operational

Do not implement yet. Ask the user to choose one.
```
