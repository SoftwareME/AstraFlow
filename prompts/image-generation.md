# Image Generation Prompt

Use this for Image2 or any image generation model when visual references are needed.

```text
Create a premium commercial-grade UI visual reference for [product type].

Audience:
- [target users]

Brand feel:
- [3-5 adjectives]

Screen:
- [screen name and purpose]

Style:
- mature, polished, production-ready
- clean information hierarchy
- realistic domain-specific data
- refined typography
- subtle borders and restrained shadows
- no decorative blobs
- no fake glassmorphism
- no childish gradients
- no text overlap
- no clipped labels

Layout:
- [navigation structure]
- [primary content area]
- [secondary panels]
- [tables/charts/cards/forms as needed]

Output:
- [desktop 16:9 / mobile screen / product shot]
- readable UI text
- realistic spacing
- looks like a real funded product
```

Save generated outputs to `assets/generated/`.
