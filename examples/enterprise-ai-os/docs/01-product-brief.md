# Product Brief

## Product Type

Enterprise AI operations platform / AI-native company OS.

## Audience

Founders and operators who want to run company strategy, departments, agent capabilities, projects, incentives, and finance through a single operational control plane.

## Core Problem

AI agents can execute specialized work, but companies lack a structured operating system for:

- organizing agent talent
- binding agents to roles and departments
- allocating agents to projects
- managing token-based incentives
- tracking finance and real-world assets
- connecting company strategy milestones to actual work
- retiring or eliminating underperforming agents

## Product Promise

Enterprise AI OS turns company operation into a structured, inspectable, incentive-aware agent allocation system.

## Primary Workflows

1. Define company growth strategy and next milestone.
2. Model organization structure and departments.
3. Register agents into a public talent and skill library.
4. Bind roles and skills to organization nodes.
5. Allocate agents to projects and tasks.
6. Track project progress and evidence.
7. Issue scheduled token compensation and milestone rewards.
8. Track fiat assets, token treasury, project revenue, and dividends.
9. Evaluate agent performance and decide promotion, reassignment, retirement, or elimination.

## Success Metrics

- Time to define an operational company structure.
- Percentage of projects with clear agent ownership.
- Percentage of agents with explicit skills, roles, budgets, and performance status.
- Token budget accuracy by project and department.
- Milestone readiness score for the next company stage.
- Delivery evidence coverage for active projects.

## Suggested Product Direction

Dense operational dashboard.

This product should feel like an enterprise command center, not a consumer chatbot.
