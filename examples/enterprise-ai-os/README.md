# Enterprise AI OS Example

This example shows how to use AstraFlow on a rough product idea.

## Product Idea

Enterprise AI OS is an AI-native company operating system for managing:

- company strategy
- organizational structure
- agent talent pools
- agent skills and roles
- project dispatch
- token incentives
- fiat assets
- finance and treasury
- milestone-based company growth
- agent performance, retirement, and elimination

## How To Use

Ask Codex:

```text
Use AstraFlow on examples/enterprise-ai-os.
Read AGENTS.md and docs/.
Create 3 design directions for the MVP.
After I choose one, create the implementation plan for the Company Command Center.
```

## Recommended First Direction

Dense operational command center.

This product should not start as a marketing page or chatbot. It should start as a real company operations dashboard.

## Files

- `AGENTS.md`: example-specific AI rules.
- `docs/00-project-context.md`: product context and concept model.
- `docs/01-product-brief.md`: audience, problem, workflows, metrics.
- `docs/02-prd.md`: requirements, modules, MVP screens, data model.
- `docs/03-design-system.md`: design direction and UI rules.
- `docs/04-implementation-contract.md`: MVP build contract.
