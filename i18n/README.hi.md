# AstraFlow

[मुख्य README पर वापस जाएं](../README.md)

**AstraFlow एक AI-native project workflow blueprint है, जो शुरुआती विचारों को designed, tested और deployable software में बदलता है।**

यह teams को product planning से PRD, design system, AI-generated visual direction, Figma handoff, frontend implementation, visual QA, automated testing, iterative fixes और deployment तक ले जाने में मदद करता है।

## AstraFlow क्यों इस्तेमाल करें

AI tools शक्तिशाली हैं, लेकिन vague prompts अक्सर vague products बनाते हैं। AstraFlow AI agents को stable project context, design rules, acceptance criteria और quality gates देता है ताकि वे भरोसेमंद तरीके से काम कर सकें।

## यह किन समस्याओं को हल करता है

- अस्पष्ट product requirements
- demo जैसी AI-generated UI
- design और engineering के बीच gap
- AI agents का context खोना
- loading, empty, error, disabled जैसी states की कमी
- UI overlap, clipped text और कमजोर responsive behavior
- idea से Figma, code, QA और deployment तक unstable workflow

## Core Workflow

```text
idea -> PRD -> design direction -> design system -> AI visuals -> Figma handoff -> frontend -> visual QA -> tests -> fixes -> deployment
```

## Quick Start

1. `docs/00-project-context.md` भरें।
2. `docs/01-product-brief.md` भरें।
3. अपने AI agent से `AGENTS.md` और `docs/` folder पढ़ने को कहें।
4. उससे 3 design directions propose करवाएं।
5. एक direction चुनें।
6. design system, implementation, QA, tests और deployment readiness तक आगे बढ़ें।
